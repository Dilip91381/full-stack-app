# 🎓 Student Manager App

A simple and clean full-stack web application built to manage student records. This project allows users to easily add and track students with their names and ages through a modern, responsive user interface.

## 🛠️ Tech Stack

This project is built using the **MERN** stack:
* **Frontend:** React.js, Axios, CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Version Control:** Git & GitHub

## ✨ Features

* **Add Students:** Easily input a student's name and age via a clean form interface.
* **Responsive UI:** A modern, centered card layout with a gradient background.
* **RESTful API:** Backend built with Express to handle data creation and storage.
* **Database Integration:** Connects to MongoDB to persist student records.
